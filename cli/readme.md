# ink-cli-app

To install dependencies:

```bash
bun install
```
To run (dev mode):

```bash
bun run start:dev
```

To generate the bundle:
```bash
bun run build
```

To run (from bundle):

```bash
bun run start
```

This project was created using `bun init` in bun v1.2.9. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
